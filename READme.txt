# Local Food Wastage Management System

## What is this?
This is a simple web app that helps match places with extra food to places that need food, so we waste less food.

## What can you do?
- See all food donations and filter by location, provider, or type.
- Add, update, or delete food providers, receivers, listings, and claims.
- Generate reports to see useful info like how many providers are in each city.
